Avec Debitum, vous pouvez suivre toutes sortes de reconnaissances de dette, qu'il s'agisse d'argent ou d'objets prêtés. Ainsi, vous n'oublierez plus jamais si votre ami vous a déjà rendu ce livre ou vous disputez sur la somme que vous devez à votre collègue pour le café.

Le cas d'utilisation prévu est la gestion des (petites) dettes personnelles et des objets prêtés. Pour garder les choses simples, les choses comme les intérêts, les échéances, les frais, etc. ne feront pas partie intégrante de l'application (bien que la description puisse être utilisée à cette fin, mais je recommande généralement d'utiliser des applications plus spécialisées pour faire ceci).

Vous pouvez sauvegarder et restaurer la base de données où les personnes et les transactions sont stockées.

Aucun service en ligne n'est impliqué, vos données sont donc sauvegardées uniquement sur votre appareil (en ce qui concerne cette application).

<i>Mots-clés : dette, emprunt, prêt, IOU, debt, lend</i>
