Track owed debts, be it money or lent items. Did your friend already give you that book back, and how much do you owe for coffee?

Everything is only saved on your device (as far as the app is concerned).

Usage

Creating Transactions

Track either lent money or items by creating a transaction using the red button. Enter amount, description or date.

Transaction Lists

Choose transaction views at the bottom:

People

Transactions summarized by person, ordered by last transaction date

  • Tap a row to show money and items for that person
  • Create persons from the menu here, or with the "+" button while adding a new transaction
Money

Shows only monetary transactions, by date (descending). Long-tap the initial one and select the "Repay" menu to create a matching repayment. Create a new one when someone repays you.

Items

Shows only lent items, by date (descending). Long-press choose "Mark as returned" for returned, unreturned, or all items.

Long-press any row for its contextual menu. Edit, delete items or select multiple rows this way. In the "Person"- and "Money" list it shows the sum of all selected rows.

Linking Contacts

"Edit person" allows linking existing contacts to persons. Click "No linked contact" and choose one to reuse contact pictures.

Backup and restore

Back up and restore the database (of all transactions and persons) from the settings. Uninstalling the app (or clearing its data) removes these files its app data folder!

Permissions

Read contacts

(Optionally) used to link contact list entries to persons.
