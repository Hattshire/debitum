Med Debitum kan du holde øye med flere forskjellige verdier og varer. På denne måten glemmer du aldri hvorvidt din venn allerede ga deg tilbake den boken, eller trenger å lure på hvor mye du skylder din kollega for kaffe.

Det er ingen nettbaserte tjenester involvert, så din data lagres kun på din enhet (sålangt programmet er involvert).

Bruk

Opprettelse av overføringer

For å bite seg merke i bortlånte penger eller ting, oppretter du en overføring ved bruk av den røde handlingsknappen. Etterpå kan du skrive inn detaljer som beløp, beskrivelse eller dato, og bytte mellom en penge- eller en gjenstandsbasert overføring (bruk bryteren til høyre for feltet for beløp).

Overføringsliste

Velg forskjellige visninger på bunnen:

Personer

Overføringer vises per person, sortert etter siste overføringsdato

  • Trykk på en rad for å filtrere penger, og ting tilknyttet den personen
  • Opprett personer fra menyen her, eller ved bruk av «+»-knappen mens du legger til en ny overføring.
Penger

Viser kun finansielle overføringer, etter dato (synkende). Lang-trykk på den opprinnelige overføringen og velg menyelementet for tilbakebetaling, som oppretter en samsvarende tilbakebetaling for deg. Hvorfor det ikke finnes noen «marker som returnert»-funksjonalitet (som den for gjenstander) er forklart her.

Gjenstander

Shows only lent items, ordered by date (descending). Items can be marked as returned by long-pressing and choosing the "mark item as returned" button. The list can then be filtered for returned/unreturned/all items.

You can get a context menu in every list by long-pressing a row. From there you can edit or delete items (also multiple rows can be selected this way). In the Person- and the Money-List the sum of all selected rows is displayed in the contextual action bar.

To create a copy of an existing transaction or use one as a preset, select it and tap the "+" button. The create-transaction-dialog will be pre-filled with the values of the selected transaction.

Linking Contacts

The "Edit person" Dialog allows linking existing phone contacts to persons. Click "No linked contact" and choose one. Currently, the only effect is, that the contact's photo (if present) is shown in the "Person" list.

Backup and restore

Back up and restore the database (of all transactions and persons) from the settings. Make sure to move backup files out of the way before uninstalling the app (or clearing app data), as the backup is saved in the app's data folder on external storage, which is deleted upon uninstall/clearing app data.

Permissions

Read contacts

Used for linking phone contacts to persons. You can safely deny this permission and use Debitum without linking contacts.
