Avec Debitum, vous pouvez suivre toutes sortes de reconnaissances de dette, qu'il s'agisse d'argent ou d'objets prêtés. Ainsi, vous n'oublierez plus jamais si votre ami vous a déjà rendu ce livre ou vous disputez sur la somme que vous devez à votre collègue pour le café.

Aucun service en ligne n'est impliqué, vos données sont donc sauvegardées uniquement sur votre appareil (en ce qui concerne cette application).

Utilisation

Création de transactions

Pour prendre note de l'argent ou des articles prêtés, créez une transaction à l'aide du bouton rouge. Une boîte de dialogue apparaît, dans laquelle vous pouvez saisir des détails tels que le montant, la description ou la date et passer d'une transaction d'argent à une transaction d'article (utilisez le bouton à droite du champ de saisie du montant).

Listes des transactions

Choisissez l'affichage des transactions en bas de la page :

Personnes

Transactions résumées par personne, classées par date de dernière transaction

  • Touchez une ligne pour filtrer l'argent et les articles par cette personne
  • Créez des personnes à partir du menu ici, ou avec le bouton + en ajoutant une nouvelle transaction
Argent

Affiche uniquement les transactions monétaires, par date (en ordre décroissant). Pour créer un remboursement, appuyez longuement sur la transaction initiale et sélectionnez l'élément de menu rembourser. Ceci créera une transaction de remboursement correspondante. Pour savoir pourquoi il n'existe pas de fonctionnalité « marquer comme rendu » comme pour les objets, voir ici .

Objets

Affiche uniquement les éléments prêtés, par date (en ordre décroissant). Pour marquer les articles comme retournés, appuyez longuement sur eux et choisissez « Marquer comme rendu » dans la barre de menu. La liste peut être filtrée par objets rendus, non rendus ou tous les objets.

Appuyez longuement sur une ligne de n'importe quelle liste pour afficher son menu contextuel. Vous pouvez modifier, supprimer des éléments ou sélectionner plusieurs lignes de cette manière. Dans les listes « Personne » et « Argent », la somme de toutes les lignes sélectionnées est affichée.

To create a copy of an existing transaction or use one as a preset, select it and tap the "+" button. The create-transaction-dialog will be pre-filled with the values of the selected transaction.

Linking Contacts

The "Edit person" Dialog allows linking existing phone contacts to persons. Click "No linked contact" and choose one. Currently, the only effect is, that the contact's photo (if present) is shown in the "Person" list.

Backup and restore

Back up and restore the database (of all transactions and persons) from the settings. Make sure to move backup files out of the way before uninstalling the app (or clearing app data), as the backup is saved in the app's data folder on external storage, which is deleted upon uninstall/clearing app data.

Permissions

Read contacts

Used for linking phone contacts to persons. You can safely deny this permission and use Debitum without linking contacts.
